# exe_client

## Usage

### Linux and macOS

Using Nix is the only supported method.

```sh
nix build
```

### Windows

> WIP

You must install the dependencies:

- [CMake](https://cmake.org/download/)
- [Ninja](https://ninja-build.org/)

```sh
cmake --workflow windows-release
```
